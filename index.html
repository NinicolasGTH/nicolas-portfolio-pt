Quem sou eu

Sou um desenvolvedor full-stack brasileiro com experiência prática na construção de sistemas web completos — do modelamento do banco de dados até interfaces frontend polidas.

Minha stack abrange Python, TypeScript, JavaScript, C# e Go, com foco em design de APIs REST, arquitetura backend e desenvolvimento frontend moderno com React e Next.js.

Trabalho com conforto em todo o ciclo de desenvolvimento: modelagem de schemas, construção de APIs, integração de interfaces, automação de fluxos e deploy de aplicações containerizadas com Docker.
